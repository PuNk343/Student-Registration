# Student Registration Form

A simple Student Registration Form built using **HTML** and **JavaScript**, with basic **client-side validation** for required input fields.

## Features

- Simple and clean registration form
- Fields for First Name, Last Name, Email, and Password
- JavaScript validation to check empty fields
- Success alert on valid submission
- Beginner-friendly project structure

## Technologies Used

- HTML
- JavaScript

## Project Structure

Student-Registration-Form/
- index.html
- README.md

## How It Works

The form takes user input for registration details.  
When the user clicks the **Submit** button:

- JavaScript checks whether all fields are filled
- If any field is empty, an alert message is shown
- If all fields are filled, a success message is displayed
